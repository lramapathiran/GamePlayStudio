# GAMEPLAY STUDIO
Une combinaison secrète de 4 chiffres doit être découverte sous 5 tentatives. Chaque chiffre allant de 0 à 9! 3 modes de jeu sont proposés: le mode défenseur où l'ordinateur doit découvrir la combinaison secrète du joueur, le mode challenger où c'est cette fois le joueur qui doit découvrir la combinaison secrète et le mode Duel où tour à tour ordinateur et joueur devine la combinaison de son adversaire!

## Pré-requis

1- Télécharger l'environnement Java sur le site d'Oracle.<br>
2- Télécharger Eclipse IDE sur https://www.eclipse.org/.<br>
3- Télécharger le dossier zip ou tar.gz d'Apache log4j 2 sur https://logging.apache.org/log4j/2.x/ pour pouvoir importer la librairie ultérieurement.
   
## Installation

1- Installer Java et eclipse comme indiqué dans les pré-requis.<br>
2- Cloner le projet à partir d'https://github.com/lramapathiran/GamePlayStudio.git.<br>
3- Ouvrir le projet dans eclipse.<br>
4- Extraire les fichiers du dossier zip Apache log4j2 téléchargé dans les pré-requis.<br>
5- Importer les jars suivantes log4j-api-2.12.1.jar, log4j-core-2.12.1.jar et log4j-1.2-api-2.12.1.jar dans l'application.<br>
